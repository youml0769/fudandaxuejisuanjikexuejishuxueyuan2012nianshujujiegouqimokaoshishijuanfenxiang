1. **二维数组相关问题**：
   - 题目描述：设 W 为一个二维数组，其每个数据元素占用 4 个字节，行下标 i 从 0 到 7 ，列下标 j 从 0 到 2。

      2. **后缀算式计算**：
         - 题目描述：后缀算式 9 2 3 + - 10 2 /。
            - 该题目要求学生根据后缀表达式的规则进行计算，考察了学生对栈的应用和后缀表达式的理解。
